VEML7700 class.

int VEML7700::Initialize ( I2CDriver &  i2c)
overridevirtual

Initialize VEML7700 with basic settings.

Initialize VEML7700 Open fd and write 4 16-bit integers to appropriate registers Gain = 01 = x2 Integration time = 0000 = 100 ms Persistance = 00 = 1 Power mode = 00 = mode 1 Refresh Time = 600 ms Resolution (lx/bit) = 0.0288

Returns
None
