Detailed Description

GNU GENERAL PUBLIC LICENSE Version 3, 29 June 2007

(C) 2020-2021, Bernd Porr mail@.nosp@m.bern.nosp@m.porr..nosp@m.me.u.nosp@m.k A thin wrapper around the C++ thread model to avoid a static callback. Instead just inherit this class and overload run() which then runs in this thread. This is header-only so that it can be performed inline for max performance.
